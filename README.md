# 記憶定着と学習方法の比較実験 (Memory Retention & Learning Methods Analysis)

## 概要 (Overview)
本リポジトリは、卒業研究「記憶方法の違い（読む・書く・聞く・話す）が記憶の定着と認知負荷に及ぼす影響」のためのデータ分析コードを管理するものである。

学習者の**ワーキングメモリ容量 (WMC)** の個人差に着目し、個人の特性に応じた最適な学習方法を統計的に解明することを目的としている。

## 実験デザイン (Experimental Design)

### 目的
「読む・書く・聞く・話す」の4つの学習方法のうち、短期記憶および長期記憶（48時間後）に最も効果的な方法はどれかを検証する。また、それが個人の記憶容量（WMC）によってどう変化するかを明らかにする。

### 条件 (Conditions)
参加者内計画 (Within-subject design) を採用し、全参加者が以下の4条件を行う。
1.  **読む (Read):** 黙読する
2.  **書く (Write):** メモを取りながら覚える
3.  **聞く (Listen):** 音声を聴く
4.  **話す (Speak):** 声に出して読む

### 測定指標 (Measures)
* **短期記憶 (Short-Term):** 直後の再生テスト成績
* **長期記憶 (Long-Term):** 48時間後の再生テスト成績
* **認知負荷 (Cognitive Load):** 主観的な大変さ (NASA-TLX等を参考)
* **ワーキングメモリ容量 (WMC):** 数字スパン課題による個人の基礎能力測定
